## Examples

### Example 1: Additive Inverse

Original statement:
`Every real number has an additive inverse.`

Equivalent rewrites:

1. `All real numbers have additive inverses.`
2. `For every real number r, there is an additive inverse for r.`
3. `For every real number r, there is a real number s such that s is an additive inverse for r.`

---

### Example 2: Pots and Lids

Original statement:
`Every pot has a lid.`

Rewrites:

1. `All pots have lids.`
2. `For every pot P, there is a lid for P.`
3. `For every pot P, there is a lid L such that L is a lid for P.`

**Solution Fill-ins**

* Blank 1: `have lids`
* Blank 2: `a lid for P`
* Blank 3: `L is a lid for P`
