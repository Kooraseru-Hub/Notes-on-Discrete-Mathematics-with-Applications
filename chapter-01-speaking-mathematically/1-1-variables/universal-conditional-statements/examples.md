## Examples

### Example 1: Animals

Original statement:
`For every animal a, if a is a dog, then a is a mammal.`

Equivalent rewrites:

* **Conditional-emphasized**
  `If a is a dog, then a is a mammal.`

* **Universal-emphasized**
  `For every dog a, a is a mammal.`
  `All dogs are mammals.`

### Example 2: Real Numbers

Original statement:
`For every real number x, if x is nonzero, then x squared is positive.`

Rewrites:

1. **Conditional-emphasized**
   `If a real number is nonzero, then its square is positive.`

2. **Universal-emphasized**
   `For every nonzero real number x, x squared is positive.`

3. **Simplified universal**
   `The square of any nonzero real number is positive.`

4. **Natural language**
   `All nonzero real numbers have positive squares.`

**Solution Fill-ins**

* Blank 1: `is positive`
* Blank 2: `x squared is positive`
* Blank 3: `is a nonzero real number; x squared is positive`
* Blank 4: `positive`
* Blank 5: `positive squares` (or `squares that are positive`)
